数据库基础知识

数据库是用于存储和管理数据的系统。

关系型数据库（SQL）：
1. MySQL - 最流行的开源数据库
   - 适用于 Web 应用
   - 支持事务处理
   - 社区版免费

2. PostgreSQL - 功能强大的开源数据库
   - 支持复杂查询
   - 支持 JSON 数据类型
   - 支持向量扩展（pgvector）

3. SQLite - 轻量级嵌入式数据库
   - 无需服务器
   - 适合移动应用和小型项目

4. Oracle - 企业级商业数据库
   - 功能最全面
   - 性能优秀
   - 价格昂贵

非关系型数据库（NoSQL）：
1. MongoDB - 文档数据库
   - 存储 JSON 格式数据
   - 灵活的模式
   - 适合快速开发

2. Redis - 内存数据库
   - 极快的读写速度
   - 支持多种数据结构
   - 常用于缓存

3. Cassandra - 分布式数据库
   - 高可用性
   - 线性扩展
   - 适合大数据场景

SQL 基本操作：
- SELECT - 查询数据
- INSERT - 插入数据
- UPDATE - 更新数据
- DELETE - 删除数据
- CREATE - 创建表
- DROP - 删除表

数据库设计原则：
1. 规范化 - 减少数据冗余
2. 索引优化 - 提高查询速度
3. 事务管理 - 保证数据一致性
4. 备份策略 - 防止数据丢失
