Python 编程语言简介

Python 是一种高级编程语言，由 Guido van Rossum 于 1991 年首次发布。

主要特点：
1. 简单易学 - Python 的语法清晰简洁，适合初学者
2. 功能强大 - 拥有丰富的标准库和第三方库
3. 跨平台 - 可以在 Windows、Linux、macOS 等系统上运行
4. 应用广泛 - 用于 Web 开发、数据分析、人工智能、自动化等领域

Python 的版本：
- Python 2.x 已经停止维护
- Python 3.x 是当前主流版本，最新版本是 Python 3.12

常用的 Python 库：
- NumPy - 数值计算
- Pandas - 数据分析
- Django/Flask - Web 框架
- TensorFlow/PyTorch - 机器学习
- Requests - HTTP 请求

Python 的应用场景：
1. Web 开发 - 使用 Django 或 Flask 框架
2. 数据科学 - 数据分析和可视化
3. 机器学习 - AI 模型开发
4. 自动化脚本 - 系统管理和任务自动化
5. 游戏开发 - 使用 Pygame 库
